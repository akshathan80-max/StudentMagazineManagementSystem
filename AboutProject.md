# Nitte Nexus - Student Magazine Management System

## Overview

Nitte Nexus is a web-based Student Magazine Management System developed using ASP.NET Web Forms, C#, and SQL Server. It simplifies the process of submitting, reviewing, and managing student magazine articles.

## Technologies Used

- ASP.NET Web Forms
- C#
- SQL Server
- HTML
- CSS
- Bootstrap
- JavaScript

## Features

- Admin Login
- Student Verification using USN
- Category Management
- Student Management
- Magazine Submission
- View Submission
- Approve / Reject Submissions
- Dashboard
- Export Reports

## Workflow

1. Student enters USN.
2. System verifies the student.
3. Student submits a magazine article.
4. Submission is saved with **Pending** status.
5. Admin reviews the submission.
6. Admin approves or rejects it.
